详细描述

多媒体播放器类

实现音视频文件的播放功能，使用 FFmpeg 进行解码，支持暂停、跳转、音量控制等功能。 通过回调函数与界面交互，基于 C++11 实现，便于跨平台移植。

解码一帧后才能获取到采样率等信息，因此将初始化放到这里

解码视频相关

视频同步到音频 音视频同步，实现的原理就是，判断是否到显示此帧图像的时间了，没到则休眠5ms，然后继续判断

有音频的情况下，将视频同步到音频 跟音频的pts做对比，比视频快则做延时

没有音频或者音频设备打开失败的情况下，直接同步到外部时钟

由于解码后的数据不一定都是yuv420p，因此需要将解码后的数据统一转换成YUV420P
