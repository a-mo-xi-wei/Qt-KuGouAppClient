详细描述

管理原始视频帧的类

该类用于存储和管理原始视频帧（YUV 或 RGB 格式），支持缓冲区初始化和分量设置。

成员枚举类型说明

枚举值
FRAME_TYPE_NONE 

无类型

FRAME_TYPE_YUV420P 

平面 YUV 4:2:0，12bpp（每 2x2 Y 样本有 1 个 Cr 和 Cb 样本）

FRAME_TYPE_RGB8 

8 位 RGB

FRAME_TYPE_RGB24 

打包 RGB 8:8:8，24bpp，RGBRGB...
