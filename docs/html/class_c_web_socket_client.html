◆ sendFile()

bool sendFile ( const QString & filepath,
bool isExcludeUserInputEvents = true,
QString rootpath = "" )

发送文件(注意点：在网络中使用时一定要设置isExcludeUserInputEvents为false,这个参数是在界面中做防假卡死的)

CWebSocketClient::sendFile 发送文件

参数
isExcludeUserInputEvents是否要做防卡死，如果是在网络中处理就不用，在界面上处理要做防卡死处理，初始是要做防卡死处理的
filepath要发送的文件的完整路径
rootpath主路径，主要去除发送的文件路径，得到文件的相对路径
返回
如果文件发送成功返回真，否则返回假

< 当前发送的文件的数据大小和文件总大小
