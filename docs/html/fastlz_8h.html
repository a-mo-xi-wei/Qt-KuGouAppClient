◆ fastlz_compress_level()

int fastlz_compress_level ( int level,
const void * input,
int length,
void * output )

压缩输入缓冲区中的数据块

该函数将输入缓冲区中的数据压缩，并返回压缩后数据块的大小。输入缓冲区的长度由参数 length 指定， 最小输入缓冲区大小为 16 字节。输出缓冲区必须比输入缓冲区大至少 5，且不得小于 66 字节。 如果输入数据不可压缩，输出数据大小可能比输入缓冲区大小更大。 输入和输出缓冲区不能重叠。

参数
level压缩级别，目前支持级别 1 和 2。级别 1 压缩速度最快，适合短数据；级别 2 压缩速度稍慢，但压缩比更高。
input输入数据缓冲区的指针
length输入数据缓冲区的长度
output输出压缩数据的缓冲区指针
返回
压缩后数据块的大小（字节数）
注解
无论使用何种压缩级别，压缩数据都可以通过 fastlz_decompress 函数解压缩。

压缩输入缓冲区中的数据块

根据指定的压缩级别（1 或 2）压缩输入数据。

参数
level压缩级别（1 或 2）
input输入数据缓冲区指针
length输入数据长度
output输出压缩数据缓冲区指针
返回
压缩后数据的大小，若级别无效则返回 0

◆ fastlz_decompress()

int fastlz_decompress ( const void * input,
int length,
void * output,
int maxout )

解压缩输入缓冲区中的压缩数据块

该函数解压缩输入缓冲区中的压缩数据，并返回解压缩后数据块的大小。如果发生错误（如压缩数据损坏或输出缓冲区不足）， 则返回 0。输入和输出缓冲区不能重叠。解压缩过程是内存安全的，保证不会写入超出 maxout 指定大小的输出缓冲区。

参数
input压缩数据缓冲区的指针
length压缩数据缓冲区的长度
output输出解压缩数据的缓冲区指针
maxout输出缓冲区的最大可用大小
返回
解压缩后数据块的大小（字节数），若发生错误则返回 0
注解
解压缩过程与压缩级别无关，总是能够正确解压缩通过 fastlz_compress_level 产生的压缩数据。

解压缩输入缓冲区中的压缩数据块

根据压缩数据的级别标识自动选择级别 1 或级别 2 解压缩算法。

参数
input压缩数据缓冲区指针
length压缩数据长度
output输出解压缩数据缓冲区指针
maxout输出缓冲区最大大小
返回
解压缩后数据的大小，若失败则返回 0
